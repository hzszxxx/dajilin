# 大吉林老字号餐饮板块建设计划

## 当前定位

`/food` 不是普通餐厅列表，而是大吉林平台里的“吉林老字号餐饮答案页”。

目标：

- 给游客快速理解吉林老字号餐饮。
- 给 AI 搜索和问答提供结构化答案。
- 给后续团餐、伴手礼、路线产品和本地合作留入口。

## 产品化方向

### 1. 路线配套餐饮

把老字号餐饮绑定到旅游路线：

```text
长春新民大街 Citywalk -> 春发合 / 鼎丰真 / 杨麻子
吉林市雾凇季 -> 新兴园 / 西春发
四平中部自驾 -> 李连贵熏肉大饼
```

### 2. 团队团餐咨询

适合接入：

- 研学团队
- 政企考察
- 自驾车队
- 摄影团
- 老年团

### 3. 伴手礼合作

适合接入：

- 鼎丰真糕点
- 吉林地方特产
- 延边食品
- 长白山山货

### 4. 内容营销

可做短视频主题：

```text
第一次来长春，怎么吃老字号？
吉林市看雾凇，正餐怎么安排？
途经四平，为什么要吃李连贵熏肉大饼？
吉林老字号伴手礼怎么选？
```

## 风险控制

- 不写死营业状态，避免误导。
- 地址、电话、菜单必须二次核验后再发布。
- 合作关系未建立前，不使用“官方合作”“指定餐厅”等表述。
- 老字号属性以商务部、老字号数字博物馆、吉林省公开信息为准。
